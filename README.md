## Project Name
Micro-Commerce for Sari-Sari Stores

## One-Line Description
A B2B procurement platform connecting sari-sari stores to regional suppliers via zero-fee USDC payment rails on Stellar.

## Track
Track 2 Financial Inclusion & Everyday Payments

## Problem It Solves
Sari-sari stores in the Philippines are often limited to local suppliers and high cash-handling costs, preventing them from expanding their product catalog. This platform allows micro-retailers to order bulk goods from regional distributors and settle payments instantly using USDC, bypassing traditional banking hurdles.

## How It Uses Stellar
- **B2B Payment Rails:** Leverages Stellar for near-zero fee settlements between sari-sari stores and regional suppliers using **USDC**.
- **Transaction Building:** Uses the `ProcurementPortal` to build and submit **XDR payment transactions**, ensuring instant settlement and supplier notification.
